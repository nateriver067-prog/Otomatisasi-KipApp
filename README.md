# KECAP — KipApp Otomasi

Aplikasi desktop Windows untuk otomasi pengisian SKP pegawai BPS melalui platform KipApp.

## Download

Unduh installer terbaru di halaman **[Releases](../../releases/latest)**.

## Instalasi

1. Download `KECAP_Setup_vX.X.X.exe` dari Releases
2. Jalankan installer
3. Buka KECAP dari shortcut yang terbuat

## Update Otomatis

KECAP akan memberitahu jika ada versi baru dan dapat mengupdate dirinya sendiri secara otomatis.
